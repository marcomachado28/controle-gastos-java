# Controle de Gastos

## Problema
Muitas pessoas têm dificuldade em controlar pequenos gastos do dia a dia.

## Solução
Este sistema permite adicionar e visualizar gastos, além de calcular o total.

## Como executar
1. Compile o código:
   javac src/Main.java

2. Execute:
   java -cp src Main# controle-gastos-java
